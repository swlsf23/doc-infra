# Writing Markdown

These samples use common features you will want to regression-test.

## Lists

- Unordered items
- Another item
  - Nested detail
  - Another nested detail

1. Ordered step one
2. Ordered step two

## Emphasis

Use **bold** and *italic* sparingly. Inline `code` should render distinctly from body text.

## Blockquote

> Quotes should survive conversion without swallowing following paragraphs.

That line should remain outside the quote.
