# Conventions

## Filenames

Use lowercase kebab-case for Markdown files: `first-build.md`, not `First Build.md`.

## Headings

Start at `#` for the page title. Avoid skipping levels without reason (`#` then `###`).

## Line breaks

One sentence per line is friendly for diffs. Wrap long paragraphs intentionally.

## Links

Prefer descriptive link text instead of bare URLs in prose.
